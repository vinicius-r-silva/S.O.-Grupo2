# Simulador de algoritmos de troca de página

Dependências:
``` 
sudo apt-get install mesa-common-dev qt5-default cmake g++
``` 

Compilar:
``` 
cd .../S.O.-Grupo2/Atividade_3/QT/Simulador_Memoria/
mkdir build
cd build
cmake ../.
make
``` 

Comandos:
P(x) C A
x = pid do processo
            
C = comando
            
A = argumento em decimal (byte como unidade)
